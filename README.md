# TinyStories SLM from Scratch 📖
This project is a from-scratch implementation of a Small Language Model (SLM) in PyTorch, designed to generate creative short stories. The model is a decoder-only Transformer, similar to GPT, with approximately 30 million parameters.

It is trained on the TinyStories dataset, a collection of short stories generated by GPT-3.5 and GPT-4 using a vocabulary that a 3-4 year old can understand.

## Features ✨
Framework: Built entirely in PyTorch.

Architecture: GPT-style decoder-only Transformer.

Tokenization: Uses tiktoken with the gpt2 vocabulary.

Modern Training Techniques:

AdamW optimizer for stable training.

Cosine annealing learning rate scheduler with a linear warmup.

Mixed-precision training (bfloat16/float16) for efficiency.

Gradient accumulation to simulate larger batch sizes.

Gradient clipping to prevent exploding gradients.

## How to Run 🚀
Clone the Repository

Bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
Install Dependencies
Create a requirements.txt file with the following content:

Plaintext
torch
datasets
tiktoken
numpy
tqdm
matplotlib
Then, install the packages:

Bash
pip install -r requirements.txt
Run the Training Script
Execute the main Python file. The script will automatically download the dataset, tokenize it, and begin training.

Bash
python your_script_name.py
The script will save the best-performing model to best_model_params.pt and plot the training/validation loss curves in loss_curves.png.

## Model Architecture ⚙️
The model is configured with the following hyperparameters, resulting in ~30M parameters:

Vocabulary Size: 50,257

Embedding Dimension (n_embd): 384

Number of Layers (n_layer): 6

Number of Attention Heads (n_head): 6

Context Window (block_size): 128

Dropout Rate: 0.1

## Output
After training, the script will:

Save the best model weights to best_model_params.pt.

Generate a loss_curves.png image showing training and validation loss.

Run inference on two sample prompts and print the generated stories to the console.
